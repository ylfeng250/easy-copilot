## 核心功能亮点
### 🚀 智能内容提取
+ 自动检测并索引用户提问，无需手动操作
+ 实时监听对话更新，新提问自动加入导航列表
+ 智能清洗文本，自动过滤代码块等干扰内容

### 📍 侧边栏导航
+ 简洁优雅的侧边栏，默认隐藏，不干扰正常使用
+ 点击任意提问，瞬间跳转至对应位置
+ 平滑滚动 + 高亮反馈，定位一目了然

### 🔍 即时搜索
+ 关键词实时过滤，快速定位特定提问
+ 显示当前对话的提问总数，进度心中有数

### ⚙️ 灵活配置
+ 预置 7+ 主流 AI 平台支持，开箱即用
+ 支持自定义 CSS 选择器，适配任意 AI 平台
+ 配置导入/导出，方便备份与迁移
