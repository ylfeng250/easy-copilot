# Easy Copilot - AI Conversation Navigator

## Short Description

Quickly navigate through your AI conversations with a smart sidebar that indexes all your questions.

## Detailed Description

**Easy Copilot** is a powerful Chrome extension designed to enhance your productivity when chatting with AI assistants. It automatically indexes all your questions in a conversation and provides an elegant sidebar for quick navigation—no more endless scrolling to find that one prompt you asked earlier!

### Key Features

**🚀 Smart Content Extraction**
- Automatically detects and indexes user prompts across multiple AI platforms
- Real-time DOM monitoring keeps the navigation list always up-to-date
- Clean text extraction removes code blocks and formatting noise

**📍 Sidebar Navigation**
- Sleek, non-intrusive sidebar appears on the right side of the page
- One-click navigation to any question in your conversation
- Smooth scrolling and visual highlight feedback when jumping to content

**🔍 Instant Search**
- Real-time keyword filtering to quickly locate specific prompts
- Displays the total count of detected questions

**⚙️ Flexible Configuration**
- Pre-configured support for popular AI platforms (ChatGPT, Claude, Gemini, DeepSeek, and more)
- Custom CSS selector support for any AI platform
- Import/export configurations in JSON format
- Enable/disable rules per platform

### Supported Platforms

| Platform | Domain |
|----------|--------|
| ChatGPT | chatgpt.com |
| Claude | claude.ai |
| Gemini | gemini.google.com |
| DeepSeek | chat.deepseek.com |
| Kimi | kimi.com |
| Qwen | chat.qwen.ai |
| Tencent Yuanbao | yuanbao.tencent.com |
| Doubao | doubao.com |

### How to Use

1. Install the extension and visit any supported AI platform
2. A trigger button appears on the right side of the page
3. Click to expand the sidebar and view all your questions
4. Click any item to instantly jump to that part of the conversation

### Privacy First

- No data collection or tracking
- All configurations are stored locally in your browser
- Open source and fully transparent
