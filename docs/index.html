安装指南

从源码构建并加载扩展

1

克隆仓库

git clone https://github.com/ylfeng250/easy-copilot.git
cd easy-copilot
2

安装依赖

pnpm install
3

构建项目

pnpm build

开发模式下可使用 pnpm dev

4

加载扩展

  1. 1. 打开 Chrome 浏览器，访问 chrome://extensions/
  2. 2. 开启右上角「开发者模式」
  3. 3. 点击「加载已解压的扩展程序」
  4. 4. 选择 build/chrome-mv3-prod 目录

使用指南

快速上手，提升对话效率

基本使用

  1. • 访问支持的 AI 平台（如 ChatGPT、Claude 等）
  2. • 页面右侧会出现一个触发按钮
  3. • 点击按钮展开侧边栏，查看所有提问
  4. • 点击任意提问项，自动滚动到对应位置

配置管理

  1. • 点击浏览器工具栏中的 Easy Copilot 图标
  2. • 为新的 AI 平台添加自定义选择器
  3. • 在保存前测试选择器是否有效
  4. • 导入/导出配置进行备份或迁移
